# ubuntu-setup
Bash script used to install development tools in Ubuntu environment.

## Dependencies

### apt

Apt package manager - ships with ubuntu.

### pip

Installed by the script with python tools bundle.

## Installation includes

- apt package upgrade & update
- git 
- python
  - python3
  - python3-pip
- VS Code
- C++ Tools
  - build-essential
  - gcc-9
  - gcc-10
  - g++-9
  - g++-10
  - valgrind
  - cmake
  - cmake-doc
  - cppcheck
  - clang-format
  - astyle
  - conan
- Java
  - openjdk 17 JDK
  - openjdk-17 JRE
- Misc
  - gnome-tweaks
  - curl
  - zsh
- Ubuntu Customs

## Usage

`sudo ./ubuntu-setup.sh`
